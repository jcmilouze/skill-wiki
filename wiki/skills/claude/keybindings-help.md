# keybindings-help — Personnaliser les raccourcis clavier

**Mission :** Modifier `~/.claude/keybindings.json` pour personnaliser les raccourcis clavier de Claude Code.

## Quand l'utiliser

- Rebinder une touche (ex: `ctrl+s`)
- Ajouter un chord shortcut (séquence de touches)
- Changer la touche de soumission
- Toute personnalisation de `keybindings.json`

## Fichier cible

```
~/.claude/keybindings.json
```
