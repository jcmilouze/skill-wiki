# simplify — Revue et simplification du code modifié

**Mission :** Relire le code récemment modifié pour identifier et corriger les problèmes de réutilisation, qualité et efficacité.

## Quand l'utiliser

Après une implémentation, pour nettoyer et simplifier avant livraison.

## Ce que ça fait

- Identifie les abstractions prématurées ou manquantes
- Repère les duplications
- Simplifie la logique inutilement complexe
- Améliore la lisibilité sans changer le comportement
